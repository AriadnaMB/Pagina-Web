Estructura

El conjunto de datos StudentLife contiene contiene 9 tipos de datos: datos de las aplicaciones abiertas, datos de calendario, datos de llamadas realizadas, datos de hábitos alimenticios, datos educativos, datos de EMA, sensores , datos de mensajes enviados y respuestas previas y posteriores a la encuesta con un tamaño total del dataset de 10.6 GB. En las siguientes subsecciones, presentamos la estructura de cada directorio y el formato de datos.

Los archivos de datos de cada subdirectorio de tipo de datos están organizados por participantes.
